Chapter 1 : Function

1.Review the lesson

A function is a rule that assigns to each element \( x \) in D exactly one element, called \( f(x) \) in E.

2.Guided Practice

\( D= \{1,2,3\} \) and \(E =\{a,b,c\}\). The rule is \(f(1)=a, f(2)=b, f(3)=a\)

Each element of D has been assigned to exactly one element in E. It is a function.
