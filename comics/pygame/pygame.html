import pygame
This line imports the Pygame library into your program.
Pygame is a set of Python modules designed for writing video games.
It provides functionalities for graphics, sound, and handling user input (like the keyboard and mouse).
pygame.init()
This function initializes all the imported Pygame modules.
Before you can use most of Pygameâ€™s features (like creating a window or playing sounds), they need to be initialized.
Calling pygame.init() sets everything up so your game can start running.
It returns a tuple with the number of modules initialized successfully and the number that failed.
