1. Knowledge

The function np.array() in NumPy is used to create a NumPy array (ndarray) from any kind of sequence-like object, such as: a Python list â†’ [1, 2, 3]
It takes the given object and converts it into a structured, efficient, and homogeneous array.

2. How to create an ndarray from a list

Numpy code 1 (0.05 xps)

            
1 import numpy as np
2 my_array = np.array([1,2,3])

Explain this code

Line 1: import numpy as np imports the NumPy library and gives it the alias np, which is the standard convention in Python.
Line 2: np.array([1, 2, 3]) converts the Python list [1, 2, 3] into a NumPy array (ndarray). The result is stored in the variable my_array.
