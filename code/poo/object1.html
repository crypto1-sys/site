Chapter 1 : Objects : the id function

1.Review the lesson

A Python object has an identity.

To find out the identity of an object, we use the id() function.

2.Guided practice

 
>>> L= [1,2] 
>>> id(L)
5243243243524

Code explained

  • A name : L
  • An object :5243243243524

The name L points to the object with identity 5243243243524
