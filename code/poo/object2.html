In Python, the == operator is used to compare two objects for equality.

 $ a == b
$ a__.eq__(b)

2.Guided practice

 
class Person:
    def __init__(self, name):
      self.name = name
    def __eq__(self, other):
       return self.name == other.name
$ p1=Person("Alice")
$ p2=Person("Alice")
$ print(p1==p2)

Code explained


p1 == p2
   │
   ▼
p1.__eq__(p2)
   │
   ▼
Compare: p1.name == p2.name
   │
   ▼
"Alice" == "Alice" → ✅ True
