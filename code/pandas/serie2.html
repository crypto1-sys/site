1.Review the lesson

The .array attribute of a Pandas Series provides access to the underlying ExtensionArray that holds the data. Unlike .values, which usually returns a NumPy array, .array returns a Pandas-specific array type (like IntegerArray, StringArray, etc.), which: Preserves missing values (pd.NA) Maintains nullable data types Is useful for advanced type-safe operations

2.Guided Practice


import pandas as pd
# Create a Series with nullable integers
s = pd.Series([1, 2, pd.NA], dtype="Int64")

# Print the original Series
print("Series:")
print(s)

# Use .array to access the underlying ExtensionArray
print("\nUsing .array:")
print(s.array)

# Check the type
print("\nType of .array:")
print(type(s.array))
