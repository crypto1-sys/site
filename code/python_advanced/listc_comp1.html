Chapter 1 Python : List Comprehension

What is a list comprehension ?

A list comprehension in Python is a concise way to create a list in a single line of code.

Example 1

            
numbers = []
for i in range(5):
    numbers.append(i * 2)
print(numbers)

Example 2

numbers = [i * 2 for i in range(5)]
print(numbers)
