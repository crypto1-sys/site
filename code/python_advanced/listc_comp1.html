A list comprehension in Python is a concise way to create a list in a single line of code. 


    [expression for i in sequence]

2.Guided practice

Python code 1

            
numbers = []
for i in range(5):
    numbers.append(i * 2)
print(numbers)

Python code 2

numbers = [i * 2 for i in range(5)]
print(numbers)
